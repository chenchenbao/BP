# 骗中骗

# [市区]


大雄(惊慌)：什么？你看到小叮当出车祸了？对方没事吧？

小夫：你怎么不先关心小叮当？

大雄：它是机器人，倒没事。

小夫(奸笑)：它把对方撞伤了，要20万。你汇给我吧，我帮你处理。

大雄：好的好的，我马上汇。

[提款机(上面贴有防诈骗标语)]

大雄：我到提款机了，帐号多少？

小夫（拍大雄肩膀）

大雄：阿，小夫你怎么出现了？

小夫：路上看到你，想捉弄一下。

大雄：那小叮当？

小夫：骗你的。这么多年没见，你还是一样容易被骗呀。

大雄(双脚瘫软)：我还以为只有老人会被骗...

小夫：这年头骗子多，谁都要小心。瞧，那里又有个太太，看起来就像被骗。

(大雄转头看到静香妈妈紧张的表情)

(大雄小夫合力出面阻止静香妈妈)

# [静香家客厅]

静香妈妈：大雄，今天多亏了你。

大雄(挠了挠头，表示害羞）

静香：妈，真的不用道谢啦。其实，大雄这个人最容易被骗了。

大雄(心中默想：一语中的）

静香妈妈：唉，不是这样的，我和大雄不同。我一定是痴呆了，否则精明如我怎么会受骗。

静香妈妈：亏我想说这辈子诈骗集团才不能碰我。

静香妈妈：我想都没想到有人冒充妳上司，說妳出事，要20万调解...

静香妈妈：我听到脸色发白...

静香妈妈：诈骗集团真是太奸诈了！

静香：这就是很典型的电话诈骗呀！

大雄（出面打圆场）：伯母没事，最可恶的是那些骗人的人啦。

静香妈妈：唉，这些人真是什么都说得出来！

静香：耶？真不知道如果对方想骗妈妈，会不会也打去骗爸爸呀？

(出现老阳的头象，箭头标注：就是这个人）

静香妈妈：阿，这个，骗子只有被教训的分吧。

大雄：说的是，骗子要倒楣了。伯父最近不是在教一门防诈骗课程吗？

静香：那是叫认知升级课程吧！

大雄：哈哈，我记不清楚。

静香：爸爸是先告诉大家，人类思考方式，没那么理性。我们自以为精明的决策，其实问题重重，还很容易被利用。

静香妈妈：啊是是是，然后接着再教具体的方法，提升自己决策思考品质，是吧？

静香妈妈：我看他最近常跟他的学生讨论。

大雄：......看来，我也要争取当伯父的学生才行。

静香：就是呀，好好学学。太笨的人，可不能娶我呢～

(一家人笑语连连）

# [静香家玄关]

老阳（带了客人回来）：啊？今天这么热闹，大雄也来啦？

静香妈妈：今天多亏大雄，我差点...

静香：妈，(小声说）有客人来，说了太丢脸，晚点说吧

老阳：怎么了吗？

静香妈妈：没事没事。 (对着客人)请进。

客人：伯母打扰，我是老师的学生，今天在学校讨论时间拖长了，所以上门打扰。

静香妈妈：(心中默想：怎么听着声音熟悉？）没事没事，你常和老师一起讨论吧？

静香妈妈：最近都讨论什么呀？

学生：骗术！

静香妈妈(心中默想：这么刚好）

静香妈妈（急忙对着老阳）：我们能不能旁听呀！

# [静香家客厅]

(老阳上课中)

老阳：锚定作用，就是给你喊个价，就算这价高的离谱，但已经操弄你的心理预期，你稍晚要杀价，也脱不开这个价格区间，你被锚定了。

(锚定作用图片)

静香妈妈：阿，上次买的那套桌子，中的就是这个计。

静香：还有，上次我去银行买基金，理财经理开口就说，人家都是买个30来万的，害我都不好意思买少。

大雄：阿，我的经验多的数不清了。

老阳：以偏概全，会被自己的经验左右，认定全世界都是这样，或放大小概率事件的机率。

老阳：老妈上次给我买了避雷保险，就是这类。因为事件煽动，放大被雷劈的机率。

老阳：其实意外险就有保障，多买，还是被忽悠了。

静香妈妈(害羞)

静香：我倒不太会被这骗。

大雄：你每次被主管数落一句，你就会否定你所有的优点，无限自责，就是以偏概全。

静香：你倒会想。

大雄：当然，每次哄你还是很累的。

静香(作生气的样子)

(老阳还说了许多人类错误决策的案例，大家都悔不当初，希望自己早点听到这课就好了）


# [静香家玄关]

学生：今天收获太大了，老师我也会全力配合，好好搜集课程相关案例的。

静香妈妈：真是好学生。

学生：伯母也请好好学习。

静香妈妈：(内心默想)咦？怎么这样说？

(学生离开后)静香妈妈：(对老阳)我今天就差点被骗......。

老阳：我知道呀。

静香妈妈：咦？

老阳：这是我让学生试验看看，没想到你一下就上当。

静香妈妈：(抓狂)

(静香、大雄赶紧出面劝解）
